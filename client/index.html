Info:

This was built using JSMess, a port of the MESS emulator to JavaScript using Emscripten. More info can be found at the link above.

Controls for Double Dragon (MegaDrive version):

Since JSMESS doesn't recognize certain keyboards properly, use the tab key to open up the settings menu, then choose "Input (this System) and assign keys to ones on your system.
